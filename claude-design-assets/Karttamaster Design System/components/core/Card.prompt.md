Surface container — card bg, default border, large radius. The shell for modals, dropdowns, panel sections.

```jsx
<Card elevation="modal" padding="lg">…</Card>
<Card elevation="dropdown" padding="sm">…</Card>
```

- `elevation`: `flat` / `dropdown` / `modal` (maps to the shadow tokens).
- `padding`: `none` / `sm` / `md` / `lg`.
